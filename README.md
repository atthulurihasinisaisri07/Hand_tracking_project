# ✋ Hand Gesture Recognition Project

This project is a real-time hand gesture recognition system built using Python, OpenCV, and MediaPipe.

## 🚀 Features
- Detects hand landmarks using webcam
- Counts fingers (1–5)
- Detects wave gesture 👋
- Displays images based on gestures
- Smooth output to reduce flickering

## 🛠️ Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## ▶️ How to Run
1. Install libraries:
pip install opencv-python mediapipe numpy

2. Run the program:
python hand_tracker.py
